# Gravity Clusterization

Интерактивное Streamlit-приложение для сравнения двух подходов к кластеризации:

- кастомный `Gravity` (плотностная идея: KDE -> моды -> слияние -> назначение)
- `DBSCAN` из `scikit-learn`

Проект ориентирован на эксперименты с 2D и N-мерными данными.

## Важно

Проект **наполовину захардкожен**: часть решений сделана быстро в экспериментальном стиле. Это осознанно, чтобы быстрее проверять гипотезы.

## Что умеет

- Генерация синтетических данных:
  - `Clustered Gaussian`
  - `Multidimensional Gaussian` (по умолчанию 5D)
  - `Concentric Circles`
  - `Spiral`
  - `Uniform Noise`
  - `Single circle`
- Загрузка данных из CSV через `From file`:
  - Drag'n'drop (`file_uploader`)
  - путь к файлу
  - выбор разделителя (`,`, `;`, `tab`)
  - выбор числовых колонок для кластеризации
- Gravity-кластеризация в N-мерном пространстве
- DBSCAN с опциональным масштабированием признаков (`StandardScaler`)
- Подсказка `recommended eps` для DBSCAN (по k-distance)
- Визуализация сравнения методов
  - для N-D данных используется проекция на первые 2 признака
- Метрики качества:
  - `silhouette`
  - `davies_bouldin`
- Экспорт результатов:
  - `cluster_assignments.csv` (метка кластера для каждой точки по обоим алгоритмам)
- Кнопка `Поиграться с параметрами`:
  - запускает перебор параметров Gravity и DBSCAN
  - выгружает `parameter_sweep_results.csv`

## Идея алгоритма Gravity

В текущей реализации сохраняется исходная идея плотностной кластеризации:

1. Оценить локальную плотность вокруг точек (гауссово ядро)
2. Поднять точки к локальным максимумам (mean-shift style)
3. Объединить близкие моды (`merge_radius`)
4. Назначить каждую точку ближайшему центру

## Формат CSV для `From file`

- Первая строка: заголовки колонок
- Для кластеризации используются только числовые колонки
- Пустые/нечисловые значения в колонке делают её непригодной для выбора

Пример:

```csv
x,y,z,label
1.2,3.4,0.7,A
1.0,3.1,0.6,B
4.8,0.2,2.1,C
```

В этом примере для анализа доступны `x`, `y`, `z`.

## Выгрузка результатов

### 1) Текущая кластеризация

Кнопка: `Скачать результаты кластеризации (CSV)`

Файл `cluster_assignments.csv` содержит:

- `row_id`
- признаки точки
- `gravity_cluster`
- `dbscan_cluster`

### 2) Перебор параметров

Кнопка: `Поиграться с параметрами`

После завершения доступна кнопка выгрузки `parameter_sweep_results.csv` со столбцами:

- `algorithm`
- `sigma`, `merge_radius` (Gravity)
- `eps`, `min_samples` (DBSCAN)
- `time_sec`
- `clusters`
- `noise_points`
- `silhouette`
- `davies_bouldin`
- `converged`, `iterations` (Gravity)

## Ограничения

- Визуализация всегда 2D: для N-мерных данных это проекция на первые две оси.
- Качество DBSCAN на N-D данных сильно зависит от масштаба признаков и `eps`.
- Перебор параметров — локальный и небольшой, это инструмент для быстрого ресерча, не AutoML.
